Enterprise UX Case Study

Elevating Global Project Oversight

A deep-dive into the design of WSP's centralized project management dashboard, balancing high-density data with executive clarity.

Context

The Challenge of Scale

WSP handles thousands of simultaneous infrastructure projects across the globe. Previous tools were fragmented, leading to "data silos" where executives lacked a single source of truth for budget health, project timelines, and risk assessments.
CORE OBJECTIVES
• Centralize data from 420+ global projects.
• Provide instantaneous "at-a-glance" health metrics.
• Reduce time-to-decision for high-risk initiatives.
• Create a unified, consistent brand experience.
Solution Part 1

The Executive Command Center

The primary dashboard was designed to provide immediate clarity. I utilized high-level KPI cards with micro-trends, a dynamic project distribution chart, and a geo-spatial map to visualize global data flow.

Figure 1.0: Main Dashboard view showing KPI consolidation and project health distribution.

User Centricity

Understanding the Stakeholders

For this project, I identified two primary personas that interact with the dashboard in diametrically opposed ways. Designing for both required a flexible "Progressive Disclosure" strategy.
👩🏻‍💼

Sarah Jenkins

Product Lead
  • Needs immediate "High-Level" project health stats.
  • Requires quick export of "Professional Summaries".
  • Focuses on Budget Utilization vs Team Velocity.
👨🏼‍💻

David Ross

Operations Manager
  • Needs to deep-dive into "Task-Level" blocking issues.
  • Requires granular filtering by Department and Budget.
  • Focuses on Resource Management & Risk Mitigation.
Solution Part 2

Granular Control & Workflow

Moving from the overview to the daily workflow, I designed a robust project list view that allows for heavy filtering and inline status updates, ensuring that "David" can find any specific project within three clicks.

Contextual Filtering

The sidebar filtering component allows users to toggle visibility based on project status (Active, On Hold, etc.), Health Score ranges, and specific Business Units like Logistics or IT.
Innovation

AI-Driven Executive Briefings

To handle the sheer volume of data, I integrated an "AI Agent" concept that generates daily executive summaries. This identifies risks before they become critical, providing human-readable insights alongside raw data.

Figure 2.0: AI-generated daily briefings highlighting delivery momentum and emerging risks.

Identity

Visual Language

The visual identity is built around WSP's bold brand colors, using the signature red sparingly for high-impact actions and alert states. The typography (Inter) was chosen for its exceptional legibility in data-dense interfaces.
#ED2024 Primary Brand Red
#2563EB Safe/Track State
#D97706 Warning/At-Risk
#1A1A1A Text & UI Main
